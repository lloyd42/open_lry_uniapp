# open_lry_uniapp
采用uniapp开发的跨平台应用，感谢 lry api 提供的数据，本项目仅供学习使用。
## 功能
目前实现实现的功能如下：
- 暗黑模式
- 中英文切换（未全字段）
- 小说、漫画、视频模块
- 搜索（仅标题）
